Quick start

Up and running in four steps.

Install the gateway

The backend server that runs tools, manages sessions, and orchestrates the AI agent.

npm install -g @jait/gateway

Start the gateway

jait

Starts on http://localhost:8000 by default. Use jait --port 9000 to change.

Connect your client

Open the web app, desktop app, or mobile app and go to Settings → Gateway connection to enter your gateway's IP or domain.

Start building

Chat with the AI, use tools, and let Jait handle the rest.

Docker setup

For production or VPS deployments, Docker is the easiest way to run the full stack.

docker-compose.yml

services:
  gateway:
    image: ghcr.io/jakobwl/jait-gateway:latest
    ports:
      - "8000:8000"
    volumes:
      - jait-data:/data
    environment:
      - NODE_ENV=production
    restart: unless-stopped

  web:
    image: ghcr.io/jakobwl/jait-web:latest
    ports:
      - "3000:3000"
    depends_on:
      - gateway
    restart: unless-stopped

volumes:
  jait-data:

Run it

docker compose up -d

The web UI will be available at http://localhost:3000 and the gateway API at http://localhost:8000.
